Proyecto: IW-BI-06 - Juego de Tronos
URI del repositorio de GitHub: https://github.com/castrillejo/Juego_de_tronos.git

Pasos para ejecutar:
1. Crear y activar un entorno virtual.
   - Linux/macOS: `python3 -m venv venv && source venv/bin/activate`
   - Windows: `python -m venv venv && venv\Scripts\activate`
2. Instalar las dependencias: `pip install -r requirements.txt`
3. Iniciar el servidor: `python manage.py runserver`

El proyecto ya incluye un conjunto de datos de prueba en la base de datos `db.sqlite3`.
